# Cash Calculator

A simple cash calculator that calculates the total value of currency notes.

## Features
- Calculate ₹500, ₹200, ₹100, ₹50, ₹20, ₹10 notes
- Instant total calculation
- Mobile-friendly interface
- Built using HTML, CSS, and JavaScript
